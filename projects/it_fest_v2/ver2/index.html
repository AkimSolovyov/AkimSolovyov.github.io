ТОЛЬКО ПЛАНИРУЕШЬ НАЧАТЬ СВОЮ КАРЬЕРУ В IT?

После IT Fest у тебя будет понятный пошаговый план действий для развития в сфере IT.

ФРИЛАНСЕР

IT Fest станет волшебной пилюлей для развития фриланс-карьеры.
